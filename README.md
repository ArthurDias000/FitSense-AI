# 💪 Fit Sense AI

### 🏋️ Sistema de Gerenciamento de Alunos com IA



\

---

## 📖 Sobre o Projeto

O **Fit Sense AI** é um sistema desktop desenvolvido em Python com foco no gerenciamento de alunos, ideal para academias ou controle pessoal.

Além das funcionalidades tradicionais de cadastro e autenticação, o sistema inclui um módulo de **previsão com Inteligência Artificial**, agregando valor analítico ao projeto.

---

## ✨ Funcionalidades

* 🔐 Autenticação de usuários (login)
* 📝 Cadastro de novos usuários
* 🔄 Recuperação de senha
* 🔑 Alteração de senha
* 👤 Cadastro de alunos
* 📋 Listagem e gerenciamento de alunos
* 🧑‍💼 Gerenciamento de usuários
* 🤖 Sistema de previsão com IA
* 🖥️ Interface gráfica interativa

---

## 🧰 Tecnologias Utilizadas

| Tecnologia            | Descrição            |
| --------------------- | -------------------- |
| Python 3              | Linguagem principal  |
| Tkinter               | Interface gráfica    |
| SQLite                | Banco de dados local |
| IA (Machine Learning) | Previsão de dados    |

---

## 📂 Estrutura do Projeto

```bash
fit_system/
│
├── banco.py              # Conexão com banco SQLite
├── login.py              # Tela inicial (login)
├── Cadastro.py           # Cadastro de usuários
├── RecuperarSenha.py     # Recuperação de senha
├── troca_senha.py        # Alteração de senha
├── CadastrarAluno.py     # Cadastro de alunos
├── ListaAlunos.py        # Listagem de alunos
├── tela_usuarios.py      # Gerenciamento de usuários
├── tela_previsao.py      # Previsão com IA
├── Tela_Guia.py          # Tela principal
├── ia_modelo.py          # Lógica de IA
```

---

## ⚙️ Como Executar

### 📥 Clonar o repositório

```bash
git clone https://github.com/seu-usuario/fit-system.git
```

### 📂 Acessar a pasta

```bash
cd fit-system
```

### ▶️ Rodar o sistema

```bash
python login.py
```
